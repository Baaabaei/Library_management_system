# راهنمای کامل استفاده از سامانه مدیریت کتابخانه

## مقدمه

این سامانه یک نرم‌افزار مدیریت کتابخانه مبتنی بر وب است که با استفاده از Flask (پایتون) در سمت سرور و HTML/CSS/JavaScript در سمت کاربر طراحی شده است. این برنامه به شما امکان مدیریت کتاب‌ها، ثبت امانت‌ها، پیگیری بازگردانی کتاب‌ها و مشاهده دیرکردها را می‌دهد.

## پیش‌نیازها

برای اجرای این برنامه نیاز دارید:
- **پایتون نسخه 3.6 یا بالاتر**
- **مرورگر وب** (کروم، فایرفاکس، اج و ...)

## نصب و راه‌اندازی

### مرحله 1: نصب پایتون
اگر پایتون روی سیستم خود ندارید، آن را از وبسایت رسمی [python.org](https://python.org) دانلود و نصب کنید.

### مرحله 2: نصب کتابخانه‌های مورد نیاز
بسته‌های مورد نیاز را با استفاده از `pip` نصب کنید:

```bash
pip install flask flask-cors
```

### مرحله 3: آماده‌سازی فایل‌ها
فایل‌های زیر را در یک پوشه مشترک قرار دهید:
- `server.py` (فایل سرور)
- `index.html` (صفحه کاربری)

ساختار پوشه باید به این شکل باشد:
```
📁 پروژه_کتابخانه/
├── server.py
├── index.html
└── 📁 data/          (این پوشه به‌طور خودکار ساخته می‌شود)
    ├── books.csv
    └── loans.csv
```

### مرحله 4: اجرای سرور
ترمینال (خط فرمان) را باز کنید و به پوشه پروژه بروید، سپس دستور زیر را اجرا کنید:

```bash
python server.py
```

خواهید دید که سرور با پیامی مشابه زیر راه‌اندازی می‌شود:
```
==================================================
📚 Library Management System - Python Server
==================================================
📁 Data directory: /path/to/your/project/data
📖 Books CSV: /path/to/your/project/data/books.csv
📋 Loans CSV: /path/to/your/project/data/loans.csv
==================================================
🚀 Starting server at http://localhost:5000
📝 Press Ctrl+C to stop the server
==================================================
```

### مرحله 5: باز کردن برنامه در مرورگر
مرورگر خود را باز کنید و به آدرس زیر بروید:
```
http://localhost:5000
```

## آشنایی با بخش‌های مختلف برنامه

برنامه دارای ۵ بخش اصلی است که با زبانه‌های بالای صفحه قابل دسترسی هستند:

### ۱. داشبورد (Dashboard)
این صفحه نمای کلی از وضعیت کتابخانه را نمایش می‌دهد:
- **تعداد عناوین کتاب**: تعداد کتاب‌های موجود در سیستم
- **کل نسخه‌ها**: تعداد کل نسخه‌های کتاب
- **امانت فعال**: تعداد کتاب‌هایی که در حال حاضر به امانت رفته‌اند
- **تحویل داده شده**: تعداد کتاب‌هایی که بازگردانده شده‌اند
- **دیرکرد**: تعداد امانت‌هایی که بیش از ۲ هفته از موعد بازگردانی گذشته است

همچنین دو جدول در این بخش وجود دارد:
- **امانت‌های فعال**: آخرین کتاب‌هایی که به امانت رفته‌اند
- **آخرین تحویل‌ها**: کتاب‌هایی که به‌تازگی بازگردانده شده‌اند

### ۲. فهرست کتب (Books)
در این بخش می‌توانید:
- **مشاهده تمام کتاب‌ها** با اطلاعات کامل
- **جستجوی کتاب** با استفاده از عنوان، نویسنده یا کد
- **فیلتر کردن** بر اساس کتابخانه (اصلی یا دانشکده برق)
- **افزودن کتاب جدید** با کلیک روی دکمه "افزودن کتاب"
- **ویرایش کتاب** با کلیک روی دکمه "ویرایش"
- **حذف کتاب** با کلیک روی دکمه "🗑"

اطلاعات نمایش داده شده برای هر کتاب:
- عنوان کتاب
- نویسنده/مؤلف
- کد کنگره
- کتابخانه محل نگهداری
- تعداد کل نسخه‌ها
- تعداد نسخه‌های موجود

### ۳. امانت‌ها (Loans)
این بخش مدیریت امانت کتاب را انجام می‌دهد:
- **ثبت امانت جدید** با کلیک روی "ثبت امانت"
- **مشاهده تمام امانت‌ها** با جزئیات کامل
- **فیلتر کردن** بر اساس وضعیت (فعال/تحویل داده شده) و کتابخانه
- **جستجو** در نام، نشان یا عنوان کتاب
- **ثبت تحویل کتاب** با کلیک روی دکمه "تحویل"
- **ویرایش رکورد** با کلیک روی "✏"
- **حذف رکورد** با کلیک روی "🗑"

### ۴. دیرکردها (Overdue)
در این بخش امانت‌هایی که بیش از ۲ هفته از تاریخ امانت آنها گذشته و هنوز بازگردانده نشده‌اند، نمایش داده می‌شود:
- **تعداد روزهای دیرکرد** به‌صورت خودکار محاسبه می‌شود
- امکان **فیلتر بر اساس کتابخانه**
- امکان **ثبت تحویل** مستقیم از این بخش

> **نکته مهم**: تاریخ‌ها بر اساس تقویم شمسی (جلالی) محاسبه می‌شوند. سیستم تاریخ سیستم عامل شما را مبنای محاسبه قرار می‌دهد.

### ۵. جستجو (Search)
یک جستجوی جامع در کل داده‌های سیستم:
- جستجو در **عنوان کتاب، نویسنده، کد کنگره**
- جستجو در **همه فیلدهای امانت‌ها** (نشان، نام، عنوان کتاب و ...)
- نتایج به‌صورت دسته‌بندی شده نمایش داده می‌شوند

## راهنمای عملیات اصلی

### افزودن کتاب جدید
1. به بخش **فهرست کتب** بروید
2. روی دکمه **"افزودن کتاب"** کلیک کنید
3. فرم باز شده را تکمیل کنید:
   - **عنوان کتاب** (اجباری)
   - **نویسنده/مؤلف**
   - **کد کنگره** (اختیاری)
   - **تعداد نسخه**
   - **کتابخانه** (اصلی یا دانشکده برق)
4. روی **"ذخیره"** کلیک کنید

### ثبت امانت جدید
1. به بخش **امانت‌ها** بروید
2. روی دکمه **"ثبت امانت"** کلیک کنید
3. فرم باز شده را تکمیل کنید:
   - **نشان** (نام خانوادگی - اجباری)
   - **نام** (اجباری)
   - **درجه** (مثلاً: سال یکم، سرباز)
   - **قسمت** (نام گروهان یا بخش)
   - **شماره پرسنلی/دانشجویی**
   - **شماره تماس**
   - **عنوان کتاب** (اجباری - با تایپ کردن، لیست کتاب‌های موجود نمایش داده می‌شود)
   - **کتابخانه** (به‌طور خودکار با انتخاب کتاب پر می‌شود)
   - **کد کنگره** (اختیاری)
   - **تاریخ امانت** (اجباری - به فرمت ۱۴۰۴/۰۷/۰۲)
   - **تاریخ بازگردانی** (در زمان ثبت امانت خالی بگذارید)
4. روی **"ذخیره"** کلیک کنید

### ثبت تحویل کتاب
برای ثبت بازگردانی کتاب، سه راه وجود دارد:
1. در بخش **امانت‌ها**، روی دکمه **"تحویل"** کنار رکورد مربوطه کلیک کنید
2. در بخش **دیرکردها**، روی دکمه **"تحویل"** کلیک کنید
3. در پنجره باز شده، **تاریخ بازگردانی** را وارد کنید و روی **"تأیید تحویل"** کلیک کنید

### ویرایش اطلاعات
برای ویرایش هر رکورد (کتاب یا امانت):
1. روی دکمه **"✏ ویرایش"** کلیک کنید
2. تغییرات مورد نظر را اعمال کنید
3. روی **"ذخیره"** کلیک کنید

### حذف رکورد
برای حذف یک کتاب یا امانت:
1. روی دکمه **"🗑"** کلیک کنید
2. در پیام تأیید، **"OK"** را بزنید

> **هشدار**: حذف رکوردها غیرقابل بازگشت است. برای کتاب‌هایی که امانت دارند، ابتدا باید تحویل آنها ثبت شود.

## قالب‌های تاریخ

برنامه از تقویم شمسی (جلالی) استفاده می‌کند. قالب صحیح تاریخ به این صورت است:
```
سال/ماه/روز
```
مثال: `۱۴۰۴/۰۷/۰۲`

شما می‌توانید از خط تیره نیز استفاده کنید:
```
سال-ماه-روز
```
مثال: `۱۴۰۴-۰۷-۰۲`

## مدیریت کتابخانه‌ها

سیستم از دو کتابخانه پشتیبانی می‌کند:
1. **کتابخانه اصلی** - کتاب‌های عمومی
2. **دانشکده برق** - کتاب‌های تخصصی

هر کتاب و هر امانت به یک کتابخانه تعلق دارد. این امکان به شما اجازه می‌دهد:
- کتاب‌ها را بر اساس کتابخانه فیلتر کنید
- امانت‌ها را بر اساس کتابخانه تفکیک کنید
- دیرکردها را برای هر کتابخانه به‌صورت جداگانه مشاهده کنید

## واردات و صادرات داده‌ها

### صادرات (Export)
برای خروجی گرفتن از داده‌ها:
1. در مرورگر به آدرس زیر بروید:
   ```
   http://localhost:5000/api/export/csv?type=all
   ```
2. فایل‌های CSV در پوشه `data` با نام‌های `export_books.csv` و `export_loans.csv` ساخته می‌شوند
3. برای دانلود هر فایل:
   ```
   http://localhost:5000/data/export_books.csv
   http://localhost:5000/data/export_loans.csv
   ```

### واردات (Import)
برای وارد کردن داده‌ها از فایل CSV:
1. از طریق مرورگر یا ابزارهایی مانند Postman، درخواست POST به آدرس زیر بفرستید:
   ```
   http://localhost:5000/api/import/csv
   ```
2. فایل‌های CSV را با کلیدهای `books` و `loans` ارسال کنید

> **نکته**: فایل‌های CSV باید با هدرهای استاندارد سیستم همخوانی داشته باشند.

## عیب‌یابی (Troubleshooting)

### خطا: "اتصال به سرور برقرار نشد"
- مطمئن شوید سرور با دستور `python server.py` اجرا شده است
- بررسی کنید که مرورگر به آدرس `http://localhost:5000` متصل است
- اگر از فایل `index.html` به‌صورت مستقیم (double-click) استفاده می‌کنید، برنامه به‌طور خودکار به `http://localhost:5000` متصل می‌شود

### خطا: "ModuleNotFoundError: No module named 'flask'"
کتابخانه‌های مورد نیاز را نصب کنید:
```bash
pip install flask flask-cors
```

### خطا: پورت ۵۰۰۰ قبلاً استفاده می‌شود
پورت دیگری را در فایل `server.py` تنظیم کنید:
```python
app.run(host='0.0.0.0', port=5001, debug=True)  # تغییر پورت به ۵۰۰۱
```
سپس در مرورگر به `http://localhost:5001` بروید.

### تاریخ‌ها به‌درستی محاسبه نمی‌شوند
- مطمئن شوید تاریخ‌ها به فرمت صحیح (سال/ماه/روز) وارد شده‌اند
- تاریخ سیستم عامل خود را بررسی کنید

## نکات پیشرفته

### ذخیره‌سازی داده‌ها
داده‌ها در فایل‌های CSV در پوشه `data` ذخیره می‌شوند:
- `books.csv`: اطلاعات کتاب‌ها
- `loans.csv`: اطلاعات امانت‌ها

می‌توانید مستقیماً این فایل‌ها را با نرم‌افزارهایی مثل Excel باز کنید و ویرایش کنید، اما بهتر است تغییرات را از طریق خود برنامه انجام دهید.

### پشتیبان‌گیری
برای پشتیبان‌گیری، کافی است پوشه `data` را کپی کنید یا از دکمه Export استفاده کنید.

### اجرا در شبکه محلی
برای اجرا در شبکه محلی و دسترسی از دستگاه‌های دیگر:
1. IP دستگاه خود را پیدا کنید (در ویندوز: `ipconfig`، در لینوکس/مک: `ifconfig`)
2. در مرورگر دستگاه دیگر به آدرس زیر بروید:
   ```
   http://[IP-دستگاه]:5000
   ```

برای دریافت پشتیبانی بیشتر یا گزارش مشکلات، می‌توانید با توسعه‌دهنده برنامه تماس بگیرید.
